# Bitcoin_Project
Goal: Use an LSTM (Artificial Intelligence) to predict Bitcoin prices.

1. DATASET:
I used a file called "bitcoin.xlsx - Sheet1.csv" which contains Bitcoin 
prices from 2017 to 2019.

2. TECHNOLOGY:
- Python 3
- VS Code
- TensorFlow / Keras (for the AI brain)
- Matplotlib (for the graphs)

3. STEPS COMPLETED:
- Cleaned the data and scaled it between 0 and 1.
- Built an LSTM model with 60 days of memory.
- Trained the model for 25 epochs.
- Generated a graph showing how the AI follows real price trends.
- Predicted a future price for Bitcoin.

4. RESULTS:
The model successfully followed the historical trends of Bitcoin. 
The green line (AI) closely matches the black line (Actual price).
